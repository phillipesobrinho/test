Subject: Content not being cached

Hi there,

I’ve looked at the cURL output and can see that the response headers show Cache-Control: no-store, no-cache. This is coming from your origin server and instructs Cloudflare not to cache your content.

To allow Cloudflare to cache your pages, you have a few options:

  • Update your origin to return cache-friendly headers like Cache-Control: public, max-age=3600
  • Use a Page Rule or Cache Rule in Cloudflare to override and force caching on static content.
  • Enable “Cache Everything” (with caution if your content is dynamic)

Also, ensure you're using HTTPS and that Edge Caching is enabled under the Caching tab.

Tools I’d use: cURL, Browser Dev Tools, Cache Analytics, Page Rules

Happy to help you configure a specific rule if you send the URL you'd like cached!
